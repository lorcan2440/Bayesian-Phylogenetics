Bayesian Phylogenetic Reconstruction

Minimal educational implementation using JC69 + Metropolis-Hastings MCMC. Input aligned DNA sequences and estimate empirical posterior probabilities of topologies P(T | D).

Constraints: all sequences same length; alphabet is A/C/G/T only; minimum 3 taxa.
